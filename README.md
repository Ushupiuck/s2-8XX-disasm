A disassembly of the Simon Wai prototype of _Sonic the Hedgehog 2_ for the Sega Genesis/Mega Drive, based on the 2007 disassembly by Esrael L.G. Neto; it results in a bit-perfect ROM with the original. This is a continuation of Alex Field's disassembly, aiming for more research and usability.
